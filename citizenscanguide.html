Citizen Archivist Scanning

Use these instructions to scan and upload documents and associated metadata for the National Archives.
Version 0.1 (upload form here)

Fundamentals of scanning for access:

  • File Type: TIFF
  • File Size: generally greater than 20MB
  • Color Mode: sRGB
  • Resolution: 600 ppi at original size, 400 ppi alternative minimum
  • Periodically check your work to avoid cropping of document
  • Record all possible metadata in the form provided

Scanning Documents and Photographic prints

  • Make sure all edges of the document are visible in the scan
  • Place original on the scanner away from the sides of the glass edge
  • No part of the original item should be cut off or cropped off in the scan.
  • In the Preview window in the scanner software, make sure the crop box extends slightly beyond the edges of the original document so a small border around the original is visible
  • Do not crop photographic images, include borders or mounting board (adjust resolution for the actual image size)

Scanner Settings

  • Scan Quality and/or Mode: Use “Best” or the best quality available
  • Auto Exposure or Image Type: Use “Photo” or “Color Photo” rather than “Document” or “Color Document” even when scanning textual items or black and white photo prints.
  • Color Mode: Always use “Color” even when scanning black and white photo prints, negatives, and documents. Select at least 24-bit color. You may use 42- or 48-bit color if available. Higher bit depths will capture more color information but also increase the file size. Always choose RGB mode, never CMYK
  • Resolution: Reflective originals (textual documents, photo prints, maps, etc.)
  • Recommended: 600 ppi at original size
    • Alternate minimum: 400 ppi at original size
  • Turn OFF all sharpening, unsharp mask, descreening, smoothing, etc.

Saving the Image File

  • 24-bit sRGB TIFF at highest image quality setting
  • File Naming: Name files with only the letters of the Latin alphabet (A-Z, a-z), numbers, hyphens and underscores. Use customized names, not scanner-assigned defaults.

Metadata

  • Carefully record all metadata in the online form provided, and upload all files (if multiple pages) at once.
  • Any information besides the standard fields should go in "Notes" (e.g. caption)
  • Ask NARA staff if there is anything you are unsure about before submitting!
